## 🚀 How to Run
Since all solutions are written in Python, you can easily run them using your local Python environment.
1. Clone the repository:
   ```bash
   git clone https://github.com/anugrahtripathi/Mission-DSA.git
   ```
2. Navigate to the problem directory:
   ```bash
   cd Mission-DSA/LEETCODE
   ```
3. Run a specific file using Python:
   ```bash
   python "42-Trapping_Rain_Water.py"
   ```
   *(Note: You may need to uncomment or add driver code/test cases inside the file to see the output in your terminal, as platform solutions often just provide a class/function.)*
